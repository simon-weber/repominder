Tired of issues like these?

Repominder can help you remember to cut a release.

How it works

1: Install the GitHub App

Repominder is a GitHub App, which means you can give it access to specific repos that you manage. All kinds of repos are supported, including organization-owned and private repos.

2: Configure release style

You need to describe to Repominder how your project represents releases in git. Both dual-branch - for example, a master and release branch - and tag-based styles are supported.

3: Get notified

You'll receive an email on Sundays if any of your configured repos have changes pending release. You also get access to readme badges that show your release status: or
